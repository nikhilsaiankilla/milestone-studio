## Keyword Placement Strategy

### Meta Title (Primary)
- Location: `<title>` tag
- Character Limit: 50-60 characters
- Current: "Milestone Studio — Free Milestone Card Creator"
- Contains: "Milestone Studio", "Free", "Milestone Card Creator"

### Meta Description (Secondary)
- Location: Meta description tag
- Character Limit: 150-160 characters
- Contains: Primary + secondary keywords naturally
- Current: Emphasizes free, customization, export, social sharing

### Keywords Meta Tag (Tertiary)
- Location: Keywords array in metadata
- Usage: For search engines, less critical but good for SEO

### H1 Tags (High Priority)
- Location: Page headers
- Strategy: Include primary keyword naturally
- Example: "Create Beautiful Milestone Cards in Seconds"

### Body Content (Supporting)
- Strategy: Natural keyword integration
- Target: 1-2% keyword density
- Avoid: Keyword stuffing

### Internal Links (Navigation)
- Strategy: Link related pages with keyword anchors
- Example: "milestone card generator" links to /editor

### Image Alt Text (Accessibility + SEO)
- Strategy: Descriptive, keyword-inclusive
- Example: "Milestone card generator showing custom gradients"

## Tools for Keyword Monitoring

### Recommended Tools
1. **Google Search Console** (Free)
   - Monitor: Keyword rankings, CTR, impressions
   - Usage: Check which keywords drive traffic

2. **Ahrefs** (Paid)
   - Monitor: Keyword difficulty, search volume
   - Track: Rankings and visibility

3. **SEMrush** (Paid)
   - Competitor analysis
   - Keyword gap analysis

4. **Ubersuggest** (Affordable)
   - Keyword ideas and volume
   - Competition analysis

5. **Google Analytics** (Free)
   - Track: Organic search traffic
   - Measure: User behavior and conversions
